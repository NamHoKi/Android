# Lost and Found App (Android MiniProject)
------------
## Project info
##### - 분실물 찾기 어플입니다.
##### 1. 잃어버린 물건의 사진을 촬영
##### 2. 물건의 사진, 위치정보, 게시글(발견한 사람의 글)이 함께 등록된다.
##### 3. 물건을 잃어버린 사람은 게시물을 통해 자신의 물건이 있는지 확인하고, 자신의 물건이라면 게시글을 통해 찾아간다.
##### 4. 게시글을 올린 사람은 포인트를 얻는다.
